To upload a new version of Cassandra:

1) Create a tarball of the Cassandra code

> tar -cvzf Cassandra_V#.#.tar.gz Cassandra_V#.#

2) Log into the following website

cassandra.nd.edu/administrator

3) Upload the Cassandra tarball:

   a) Select the menu 'Content' --> 'Media'
   
   b) Double click the 'code' thumbnail
   
   c) Click 'Upload', click 'Choose Files'
   
   d) Navigate to the tarball, click 'Open'

   e) Click 'Start Upload'

4) Update the download target 

   a) Select the menu 'Extensions' --> 'Modules'

   b) Click 'Cassandra download'

   c) Edit the 'Code location' textbox

   d) Click 'Save & Close'

5) Upload the Cassandra user_guide_#.#.pdf into the documentation folder.

6) Update the User Guide link

   a) Select the menu 'Content' --> 'Articles'

   b) Click 'Documentation'

   c) Place the curser in the highlighted text 'Cassandra User Guide (pdf)'

   d) Click the 'Insert/Edit Link' toolbar button (picture of chain links)

   e) Edit the 'URL' textbox. Click 'Update'

   f) Click 'Save & Close'
